### GIN 实现一个简单的api server

### 搭建golang开发环境
- [https://golang.google.cn/dl/](https://golang.google.cn/dl/)下载相应的安装包
- 配置$GOBIN
- 配置$GOPATH, $GOPATH建src,pkg,bin文件夹
- 命令行`go version`和`go env`
- 配置$GO111MODULE
- 配置$GOPROXY

### go mod搭建项目
- go mod init project_name

### 初始化&&运行
- go mod tidy
- go run main.go
